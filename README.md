# Exploring the Interaction between Compact Binaries and their Host Galaxy
This project was the base of my Master's thesis at the University of Manchester in the 2023/2024 academic year. My suprevisor was Renè Breton and Soheb Mandhai acted as my co-suprevisor. I was part of the Jodrell Bank Centre for Astrophysics specifically the Pulsars and Time Domain Astrophysics Group.
This repository contains:
  > First and Second semester's Thesis (this is a common practice in the Integrated Master's degrees (MPhys) in the UK) in a .pdf format.
  
  > The Python code I authored to carry out the project and the relevant simulations. Only the *last* edition of the *main* code is presented here.
